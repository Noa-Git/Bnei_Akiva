## Bnei Akiva
This application is made for Israeli youth movement called 'Bnei Akiva'.
The app is aimed for Bnei Akiva's members- childern and their parents as well as the guides and other of the movement's authorities.

##### Tech stack:
* CodeIgniter 3
* PHP 5.3
* JavaScript
* HTML5
* CSS
* Bootstrap
